# spa-panel

## Project setup
```
yarn install
```
Change in the .env file the api url. If you're using laravel artisan serve, put the `http://localhost:<servePort>`

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
